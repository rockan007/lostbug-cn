# 左侧分类导航侧边栏

> **日期**: 2026-06-27
> **状态**: 待实施

## 目标

将分类从页面内网格移到左侧可折叠侧边栏，提供更便捷的导航体验。

## 布局

大屏（≥1024px）时为左右双栏布局：

```
┌──────────────────────────────────────────────────┐
│  LostBug  [☰]  搜索...               + 推荐网站   │  ← Navbar
├────────────┬─────────────────────────────────────┤
│ 侧边栏      │  主内容区                            │
│            │                                     │
│ 🔥 热门推荐 │   🔥 热门推荐 / 分类页               │
│ 设计工具 (2)│   [卡片] [卡片]                      │
│ 开发资源 (3)│   [卡片] [卡片]                      │
│ 效率工具 (1)│   ...                               │
│ 学习资源 (0)│                                     │
│ 资讯阅读 (1)│                                     │
│ 生活服务 (1)│                                     │
└────────────┴─────────────────────────────────────┘
```

小屏（<1024px）时侧边栏隐藏，通过 Navbar 的 ☰ 按钮打开 overlay/drawer。

## 组件

### Sidebar（新建 `src/components/Sidebar.tsx`）

- 客户端组件（`'use client'`），管理折叠/展开状态
- Props: `categories: { name: string; slug: string; count: number }[]`
- 展开态：显示"🔥 热门推荐"链接 + 所有分类及网站数
- 折叠态：窄条（~48px），仅显示展开按钮
- 当前路由高亮对应分类项
- 折叠状态持久化到 localStorage

### Navbar 修改

- 移除"分类 ▾"下拉按钮
- 新增 ☰ 切换按钮，点击切换侧边栏展开/折叠
- 其余元素（Logo、搜索框、+推荐网站）保持不变

### Layout 修改（`src/app/layout.tsx`）

- 从单列 `main` 改为 `div.flex` 包裹 `Sidebar` + `main`
- Navbar 保持 sticky top
- Sidebar 数据从 layout 服务端获取（categories + counts）

### HomePage 修改（`src/app/page.tsx`）

- 移除"📂 分类浏览"区域（已移至侧边栏）
- 保留"🔥 热门推荐"和"🆕 最新添加"

## 交互

- 侧边栏折叠按钮：☰ 切换展开/折叠
- 折叠态 → 展开态：侧边栏滑出，主内容区收窄
- 点击"热门推荐"：跳转 `/`
- 点击分类项：跳转 `/category/:slug`
- 当前路由高亮匹配的分类项

## 不变内容

- 分类页 `/category/[slug]` 保留
- 搜索功能保留
- 投票功能保留
- 所有 API 路由不变

## 技术约束

- 使用 Tailwind CSS v4
- Next.js 16 App Router
- 侧边栏为客户端组件，其余服务端组件
- 折叠状态用 localStorage 持久化，SSR 时默认展开
